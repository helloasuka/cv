O objectivo deste laboratório é a criação de uma página em HTML. A página deverá ter:

  1. Informação sobre si;
  2. Links (ligação a outras paginas)--‐URLs absolutos;
  3. Estar dividida em várias páginas (segundo temas, por exemplo) – a ligação a outras páginas criadas por si deverá ser conseguida através de links relativos.
A página deverá ser criada utilizando a aplicação Notepad++. A página principal deverá conter informação e uma foto. A partir da página principal poderão existir:
  1. Página com informação sobre hobbies (desporto, música, ...);
  2. Página com a informação do CV (nesta página deverá existir um link para a página da FCT/UNL);
  3. Outra informação...

Guião

Antes de começar faça um plano da página a criar (com os esquemas das várias páginas que serão dependentes da inicial).

  1. Estruturação da área para o site: 
    1. Criar uma pasta dentro da pasta Documentos para conter a página;
    2. O ficheiro da página principal deverá chamar‐se index.html;
    3. Para cada tipo de ficheiro contido no site deverá existir uma pasta adicional (por exemplo, a fotos deverão estar guardadas numa pasta de nome images)
  2. Abra o Notepad++ para a edição html directa. O ficheiro index.html será editado aí. Para a visualização do resultado abra o mesmo ficheiro no browser que prefere. Se abrir o ficheiro em vários browsers poderá verificar diferenças entre eles. Pode utilizar o documento em http://www.put.com/HTMLPrimer.html (procurar The Minimal HTML Document)  como base modificá-lo.
  3. Modifique o título do documento de forma a que fique “Página pessoal de …”. Verifique a modificação do browser.
  4. Analise a diferença entre a zona de título e a zona de corpo do documento.
  5. Defina o texto que pretende para a sua página adicionando parágrafos e mudanças de linha. Verifique a diferença entre uma coisa e a outra.
  6. Crie listas de itens (passatempos, cadeiras, ...).
  7. Experimente vários tipos de alinhamentos de texto e de imagens.
  8. No final de cada página  coloque uma barra horizontal.
  9. Experimente estilos diferentes.
  10. Insira um link para o utilizador lhe enviar um email.
  11. Utilize os acentos definidos no HTML.
